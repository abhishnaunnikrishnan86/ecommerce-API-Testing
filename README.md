# OpenCart API Automation (Postman)

## Project Description
Automated OpenCart cart APIs using Postman.

## APIs Covered
- Generate Session Token
- Add Product to Cart
- Get Cart Content
- Edit Product Quantity

## Tools Used
- Postman
- OpenCart
- GitHub

## How to Run
1. Import collection into Postman
2. Set base URL as:
   http://localhost/opencart
3. Run collection
